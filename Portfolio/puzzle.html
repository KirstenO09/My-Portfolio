Puzzle Game

Developed a fun interactive puzzle game designed to challenge problem solving skills.
Features a timer and number of moves for an enjoyable user experience.

I created an interactive puzzle game using HTML, CSS, and JavaScript, designed to challenge users while keeping them engaged. The game features a timer and number of moves counter, encouraging players to improve their performance by using the least amount of moves in the shortest period of time with each attempt. Through dynamic animations, smooth transitions, and an intuitive interface, I ensured that the game provides an enjoyable user experience. JavaScript handles the game logic, and managing the countdown timer to add an element of urgency. This project showcases my ability to combine functionality with engaging design, creating an interactive and immersive web-based game.
